# OpenAI Realtime Voice Chat on Streamlit

![UI Image](./resources/ui.png)

## Installation

```sh
uv sync
```

## Configuration

Create `.streamlit/secrets.toml` with:

```toml
OPENAI_API_KEY = "sk-..."
```

## Run

```sh
uv run streamlit run main.py
```
